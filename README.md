# Todo-list-app

A simple but sleek and responsive todo-list app built with React and pure css

## 🧩Features 

- [x] Add task to list ➕
- [x] Remove task from list ➖
- [x] Mark task as completed ✔️
- [x] Responsive and seemless design 📱
- [x] Star repo button 🌟
